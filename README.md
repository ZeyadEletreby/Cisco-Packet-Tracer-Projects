# Cisco Packet Tracer Labs

Welcome to my collection of networking labs built with Cisco Packet Tracer. This repository serves as a practical workspace for designing, configuring, and troubleshooting network topologies. 

## 📈 Progressive Complexity

The labs in this repository are designed with a **gradual learning curve**. Each project builds directly upon the concepts introduced in the previous ones, steadily increasing in scope and complexity. 

The progression moves from foundational device configurations up through advanced routing protocols and complete network designs.

### General Progression:
* **Level 1:** Basic router and switch configuration, static routing, and initial connectivity.
* **Level 2:** Switching technologies (VLANs, Trunking, Inter-VLAN routing).
* **Level 3:** Network services (DHCP, NAT, ACLs).
* **Level 4:** Dynamic routing protocols (RIPv2, Single-Area OSPF).
* **Level 5:** Comprehensive, multi-branch network topologies combining all the above concepts.

## 🛠️ Tools & Goals
* **Environment:** Cisco Packet Tracer
* **Focus:** Practical application of networking fundamentals
